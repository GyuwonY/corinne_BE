# 항해99 6기 B반 4조 실전 프로젝트

2022년 4월 22일 ~ 2022년 6월 3일

## 프로젝트 소개

코린이들을 위한 모의투자 corinne

corinne는 코인 초보자분들이 쉽고 재미있게 접할 수 있도록 다양한 기능들을 제공하는 코인 모의투자 서비스입니다.
